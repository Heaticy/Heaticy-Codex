# Changelog

## 1.0.0 — First Public Open Source Release

- Prepared Heaticy-Codex as a local-first, GitHub source release.
- Added explicit privacy documentation for local transcripts, approvals, audit logs, runtime files, cookies, and LAN exposure.
- Documented desktop browser and mobile LAN access with configurable `PORT`, `WEB_PORT`, and `HOST=0.0.0.0`.
- Added startup port checks so occupied ports are reported and replacement ports can be recommended.
- Rewrote English and Chinese README files for the public 1.0.0 release.
- Added compressed desktop and mobile screenshots under `docs/images/`.
- Updated security and contribution templates to avoid public sharing of tokens, `.env`, transcripts, private paths, sensitive logs, and screenshots with secrets.
- Added release checklist and GitHub Release notes draft.
