# Heaticy-Codex 1.0.0

First public open-source release of Heaticy-Codex.

## Highlights

- Local-first browser workspace for Codex sessions.
- Desktop browser and mobile LAN access.
- Configurable `PORT`, `WEB_PORT`, and `HOST=0.0.0.0`.
- Startup checks for occupied ports with recommended replacements.
- Clear privacy documentation: no Heaticy-Codex upload service, local transcripts and app state stay on your machine.
- English and Chinese README files.
- Public release checklist and security guidance.

## Notes

This release is intended for personal local and LAN use. Do not expose the service directly to the public internet without your own HTTPS, authentication, and network controls.
